# 🎓 Student Portfolio & Registration Form

## 📌 Project Description

This project is a beginner-friendly HTML5 project created to practice HTML fundamentals. It demonstrates the use of various HTML tags, form elements, input types, and attributes commonly used in web development.

The project contains a Student Portfolio & Registration Form that collects personal, educational, skill, and social information from users.

---

## 🚀 Features

### Personal Information

* Full Name
* Email Address
* Password
* Mobile Number
* Date of Birth
* Age
* State Selection
* Gender Selection
* About Yourself Section

### Education Details

* College Name
* Branch
* Current Year
* CGPA

### Skills Section

* HTML
* CSS
* Java
* Spring Boot
* Skill Confidence Slider
* Favorite Color Picker

### Social Information

* Portfolio Website
* LinkedIn Profile
* Technology Search Box

### Document Upload

* Resume Upload
* Profile Photo Upload

### Additional Inputs

* Preferred Joining Month
* Available Week

* Meeting Schedule
* Hidden Input Field
